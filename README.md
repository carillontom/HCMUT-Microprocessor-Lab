# HCMUT---Microprocessor-Lab
Microprocessor Lab at HCMUT projects about using assembly language in programming AVR chip and controls peripherals.
This course provides knowledge about microprocessor/microcontroller architecture, memory systems, basic peripherals, data transmission techniques, and assembly/C programming for microcontrollers.
